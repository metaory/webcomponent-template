Usage

Include the component via CDN:

<script type="module" src="https://unpkg.com/__LIB_NAME__"></script>

Use in your HTML:

<__LIB_NAME__ cols="3" gap="1rem" padding="1rem">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</__LIB_NAME__>

Attributes

  • cols: Number of columns or "auto-fit" for responsive grid
  • gap: Spacing between grid items
  • padding: Internal padding of the grid container
